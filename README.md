# msrc 是一个src漏洞挖掘信息收集辅助工具，爬取src厂家所有厂家名称，获取域名信息，fofa查询：
#### 基本命令：
```msrc  -h
  -bn string
        请出入src厂家名称（可以模糊查询）
  -d string
        shodan 查询
  -n string
        根据单个厂家名称从天眼查获取域名
  -p string
        请输入src厂家id页(1-189页)
  -r string
        请输入txt文件路径 (根据厂家名称从天眼查获取域名)
  -s string
        fofa 查询
  -w string
        请输入域名（可查询备案信息）
  ```
