# msrc 是一个src漏洞挖掘信息收集辅助工具，可以爬取src厂家所有厂家名称，然后获取域名信息，也可以查询域名信息：
#### 基本命令：
```msrc  -h
  -bn string
        请出入src厂家名称（可以模糊查询）
  -n string
        根据单个厂家名称从天眼查获取域名
  -p string
        请输入src厂家id页(1-189页)
  -r string
        请输入txt文件路径 (根据厂家名称从天眼查获取域名)
  -v    Show version information
  -w string
        请输入域名（可查询备案信息）
  ```
